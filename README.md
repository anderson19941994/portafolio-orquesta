🎸 Herencia García - Orquesta Criolla
Proyecto web profesional desarrollado para la agrupación Herencia García, enfocado en la promoción de la música criolla peruana, gestión de eventos de gala y preservación del legado cultural nacional.

🚀 Características
Diseño Elegante: Interfaz sofisticada con una paleta de colores Guinda y Dorado, inspirada en la tradición peruana.

Arquitectura Multi-página: 5 secciones totalmente funcionales (Inicio, Historia, Elenco, Galería y Contratos).

Galería Dinámica: Sistema de mural fotográfico (CSS Grid) con efectos visuales tipo overlay para presentaciones de gala.

Conversión de Clientes: Formulario de cotización optimizado y botón flotante de WhatsApp para agendamiento directo de shows.

Responsive & Semántico: Estructura basada en HTML5 semántico y diseño adaptativo para móviles.

🛠️ Tecnologías utilizadas
HTML5: Estructura semántica avanzada para SEO y accesibilidad académica.

CSS3: Uso de Custom Properties (Variables), Flexbox y CSS Grid para layouts complejos.

Google Fonts: Tipografía de alta gama combinando Playfair Display (Elegancia) y Poppins (Modernidad).

FontAwesome: Integración de iconografía musical y social media.
